# MBAI 410 - Python Programming Projects

A collection of Python programming assignments completed as part of the MBAI 410 course at Kellogg School of Management, Northwestern University.

## Overview

This repository contains nine comprehensive programming assignments demonstrating proficiency in Python, data structures, algorithms, and machine learning concepts.

## Projects

### Assignment 1: Pattern Matching
**File:** `Assignment 1 completed/match.py`

Pattern matching implementation focusing on string algorithms and basic Python fundamentals.

### Assignment 2: Core Programming Concepts
**File:** `Assignment 2/assignment_2.py`

Foundation-level programming exercises covering Python syntax, control structures, and functions.

### Assignment 3: Advanced Data Manipulation
**File:** `Assignment 3/Assignment3.py`

Advanced programming techniques and data manipulation strategies.

### Assignment 4: Graph Algorithms & Shortest Path
**Files:**
- `Assignment 4/Assignment_4.py`
- `Assignment 4/shortest_path_pseudocode.py`

Implementation of graph algorithms and shortest path solutions, demonstrating understanding of algorithmic complexity and optimization.

### Assignment 5: Text Analysis & Sentiment Classification
**Files:**
- `Assignment 5/Assignment_5.py`
- Data: `movie_reviews/`, `neg.dat`, `pos.dat`, `sorted_stoplist.txt`

### Assignment 6: Advanced Text Mining
**Files:**
- `Assignment 6/Assignment_6.py`

## Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mbai-410-python-projects.git
cd mbai-410-python-projects
```

2. Ensure Python 3.x is installed:
```bash
python --version
```

3. Navigate to specific assignment folders and run the Python files:
```bash
cd "Assignment 7/Assignment7"
python main.py
```
